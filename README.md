# Maximum-Product-SubArray-Continuous
 Code to find the maximum product among all its continuous subarrays.
 
Examples: a) Input: [5, 7, 9, 8, -9, 6, 8, 4]
                        Output: 2520
                    b) Input: [7, -9, 2, -1]
                        Output: 126
                    c) Input: [5, 7, -5, 0, 6, 7, -4, 0, 2, -2, -3, 1]
                        Output: 42
